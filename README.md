# Pokémon Team Generator

A web tool designed to generate random competitive Pokémon teams based on specific rulesets (regulations).

## Features
- **Regulation Support:** Choose between the Master Catalog or specific regulation rules.
- **Mega Evolutions:** Optional selection of one or two Mega Evolutions.
- **Smart Filtering:** Automatically prevents duplicate species (including different forms like Alola/Hisui/Paldea forms).
- **Live Data:** Fetches up-to-date information and sprites from the [PokeAPI](https://pokeapi.co/).

## How to use
1. Open the generator in your browser.
2. Select your desired regulation.
3. Toggle Mega Evolution options if needed.
4. Click "¡Elegir Pokémon!" to generate a new team.
